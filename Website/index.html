Installation Guide

Follow the correct method for your device.

📱 Android Installation

  1. Download the APK file.
  2. Open it from Downloads.
  3. Allow Install Unknown Apps if prompted.
  4. Tap Install and launch the app.

🍎 iOS – Install with AltStore

  1. Install AltStore on your computer.
  2. Connect your iPhone via USB.
  3. Open AltStore and sign in with Apple ID.
  4. Open AltStore on your device.
  5. Select “My Apps” → “+” → choose StreakSphere IPA.
  6. Wait for installation to complete.
Recommended Method

🍎 iOS – Install with Sideloadly

  1. Download and install Sideloadly.
  2. Connect your iPhone via USB.
  3. Drag and drop the IPA file into Sideloadly.
  4. Enter your Apple ID for signing.
  5. Click Start to install.
  6. Trust profile in Settings → VPN & Device Management.

ℹ️ Important iOS Notice

Apple requires apps to be signed before installation. Free Apple IDs may require re-signing every 7 days. This is an Apple platform limitation.
